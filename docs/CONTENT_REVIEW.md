# Content Review

## Deaf User Review

Before describing a hand shape as correct, collect review from Deaf users or Korean Sign Language experts.

## KSL Teacher Review

Review should confirm:

- the dictionary entry is the intended fingerspelling symbol
- learning tips are accurate
- common mistakes are not misleading
- feedback language is supportive and not overly absolute

## Dictionary Entry Check

Each lesson should eventually have:

- dictionary query
- selected dictionary entry ID
- source URL
- review status

## Correction Procedure

1. Log the issue.
2. Mark the lesson as review pending if needed.
3. Remove or soften any unverified learning tip.
4. Update dictionary mapping only after checking the source.
5. Add a regression test or content note.
